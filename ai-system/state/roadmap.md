# RGT Website - Roadmap

## Current Phase: Maintenance Mode

No new features planned. The site is complete and serving its purpose.

## Ongoing Maintenance Tasks
- **Content updates**: Update service descriptions, pricing, or contact info when RGT's business changes
- **SEO adjustments**: Monitor search rankings and tweak meta descriptions, titles, or structured data as needed
- **Security patches**: Apply minor dependency updates for security vulnerabilities (patch versions only)
- **Performance checks**: Periodic Core Web Vitals review to ensure fast load times
- **Bug fixes**: Address any reported issues with contact form or page rendering

## Not Planned
- New feature development
- Framework upgrades (Next.js, React, Tailwind)
- Database integration
- Multi-language support
- Blog or content management system
- User accounts or authentication

## Trigger for Reactivation
Development would resume only if:
- RGT significantly expands services requiring a site overhaul
- A major business pivot requires new functionality (e.g., online booking)
- Critical security vulnerability requires a framework upgrade
